# HOW TO PLAY
BRICKS		break all the bricks to win
LIVES		lose lives by hitting the bottom of the screen
		lose 3 lives and GAME OVER


# CONTROLS
ENTER		starts the game (START)
BACKSPACE	restart the game at any time (SELECT)
LEFT		moves the player left
RIGHT		moves the player right
UP		moves the player up
DOWN		moves the player down
